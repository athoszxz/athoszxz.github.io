Modelo de caixa em CSS (OverFlow)

Exemplo com overflow: visible

Esta caixa tem uma altura e uma largura. Isso significa que se houver muito conteúdo a ser exibido dentro da altura atribuída, haverá uma situação de estouro. Como o estouro está definido como visível, qualquer estouro ficará visível.



Exemplo com overflow: hidden

Esta caixa tem uma altura e uma largura. Isso significa que se houver muito conteúdo a ser exibido dentro da altura atribuída, haverá uma situação de estouro. Aqui o estouro está definido como oculto, logo, qualquer estouro não ficará visível.

Exemplo com overflow: scroll

Esta caixa tem uma altura e uma largura. Isso significa que se houver muito conteúdo a ser exibido dentro da altura atribuída, haverá uma situação de estouro. Como o estouro estiver definido como scroll, aparecerá uma barra de scroll para controlar o estouro.

Exemplo com overflow: auto

Esta caixa tem uma altura e uma largura. Isso significa que se houver muito conteúdo a ser exibido dentro da altura atribuída, haverá uma situação de estouro. Como o estouro está definido como auto, o navegador identificará automaticamente uma situação de estouro e colocará um scroll caso necessário.
