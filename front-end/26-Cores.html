Cores em CSS


Cores Hexadecimais

Uma cor hexadecimal é especificada com: #RRGGBB, onde os inteiros hexadecimais RR (vermelho), GG (verde) e BB (azul) especificam os componentes da cor. Todos os valores devem estar entre 00 e FF.

Vermelho #ff0000

Verde #00ff00

Azul #0000ff

Amarelo #ffff00

Violeta #ff00ff


Cores Hexadecimais com Transparência

Para adicionar transparência, adicione ao final dois dígitos adicionais entre 00 e FF, este corresponde a opacidade. Neste caso coloquei 80% de opacidade

Vermelho

Vermelho com Transparência #ff000080

Verde

Verde com Transparência #00ff0080

Azul

Azul com Transparência #0000ff80


Cores RGB

Uma cor RGB é especificada com: rgb(RR,GG,BB), onde os inteiros RR, GG e BB especificam os componentes da cor. Todos os valores devem estar entre 0 e 255.

Vermelho #rgb(255,0,0)

Verde #rgb(0,255,0)

Azul #rgb(0,0,255)

Cinza #rgb(192,192,192)


Cores RGB com Transparência (RGBA)

Para adicionar transparência, adicione ao final um dígito adicional entre 0 e 1, este corresponde a opacidade. Neste caso coloquei 0.5 de opacidade.

Vermelho com Transparência #rgb(255,0,0,0.5)

Verde com Transparência #rgb(0,255,0,0.5)

Azul com Transparência #rgb(0,0,255,0.5)

Cinza com Transparência #rgb(192,192,192,0.5)


Cores HSL

Uma cor HSL é especificada com: hsl(H,S,L), onde os inteiros H, S e L especificam os componentes da cor. Todos os valores devem estar entre 0 e 360, 0 e 100 e 0 e 100.

Verde #hsl(120,100%,50%)

Verde Claro #hsl(120,100%,75%)

Verde Escuro #hsl(120,100%,25%)

Verde Pastel #hsl(120,60%,70%)

Violeta #hsl(290,100%,50%)

Violeta Pastel #hsl(290,60%,70%)


Cores HSL com Transparência (HSLA)

Para adicionar transparência, adicione ao final um dígito adicional entre 0 e 1, este corresponde a opacidade. Neste caso coloquei 0.3 de opacidade.

Verde com Transparência #hsl(120,100%,50%,0.3)

Verde Claro com Transparência #hsl(120,100%,75%,0.3)

Verde Escuro com Transparência #hsl(120,100%,25%,0.3)

Verde Pastel com Transparência #hsl(120,60%,70%,0.3)

Violeta com Transparência #hsl(290,100%,50%,0.5)

Violeta Pastel com Transparência #hsl(290,60%,70%,0.3)

A Palavra chave "Cor Atual"/"currentcolor"

A palavra chave currentcolor é usada para indicar que a cor deve ser a cor atual do elemento. Por exemplo, se o elemento estiver em uma cor vermelha, o valor de cor do elemento será vermelho.

Este elemento Div é azul, usando "currentcolor" na borda faz ela ficar azul também.
